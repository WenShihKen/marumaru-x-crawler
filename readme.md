# marumaru-x 歌詞備份工具 

## 目標網站
[marumaru-x](https://www.marumaru-x.com)


## 安裝相依套件
```bash
pip install -r requirement.txt

playwright install
```

## Crawling
```bash
python song_list_crawler.py

python lyrics_crawler.py
```
